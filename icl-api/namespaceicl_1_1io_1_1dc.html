sets iso transmission speed for the wrapped dc1394 context

Parameters
camThe dc camera
mbitsmbits value (usefull might be 800 whereby 400 is default) values that do not make sense are ignored. The current libdc library documentation lists the following available speed values: 100, 200, 400, 800, 1600, 3200. But please note, that 1600 and 3200 are not supported by any camera yet. These values are only reserved for later technologies
