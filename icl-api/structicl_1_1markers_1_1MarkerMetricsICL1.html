Detailed Description

Marker metrics for "icl1" markers.

each marker has

  • a border of width 1 unit -> root regions
  • 4 child regions (cr) each of height 3 units and width 11 units
  • each child regions contains 1 to 5 child-child-regions (ccr) each round and with diameter of 1 unit
#############<- border (root region)
#           #
# #       # #
#         <-#-- child region cr 0
#############
#           #
# #   #   # #
#         <-#-- child region cr 1
#############
#           #
# # #   # # #
#         <-#-- child region cr 2
#############
#           #
# # # # # # #
# ^        <-#-- child region cr 3
##|##########
  |
  child child region ccr 0

The ccr's are distributed towards the edges, only in case of a single ccr within a cr, it is placed in the middle.

The whole marker structure becomes 13 x 17 units. These are mapped linearily to the actual marker size in mm;
