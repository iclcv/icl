Detailed Description

template<class Scalar, int CAPACITY = 4, int SF = 1, int ALLOC_CHUNK_SIZE = 1024, class VECTOR_TYPE = FixedColVector<Scalar,2>>
struct icl::math::QuadTree< Scalar, CAPACITY, SF, ALLOC_CHUNK_SIZE, VECTOR_TYPE >::Allocator

Inernally used block allocator.

The allocator allocates ALLOC_CHUNK_SIZE*4 Node instances at once and automatically frees all data at destruction time
